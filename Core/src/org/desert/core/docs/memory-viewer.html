6. Memory Viewer

This window displays the contents of a declared memory. The contents are editable by double-clicking the corresponding table cell. The base of the shown content is changeable.

Base .  the base of the contents can be changed here. The adresses are always shown with hexadecimal base. „Bases“)

Go to .  jumps to the given adress (hexadecimal).

Reset .  sets the contents of the corresponding memory to its initial values.

Load .  loads another content table.

Save .  saves the current content.

Further information about the structure of memory files are given in the documentation.
